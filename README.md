# BrodCastWeb | Ingeniería Digital de Élite

Este es un proyecto de landing page premium diseñado para **BrodCastWeb**, un laboratorio de ingeniería digital especializado en ecosistemas web, productos mobile y agentes de IA.

## Características
- **Diseño Ultra-Premium:** Estética oscura con toques de neón green y glassmorphism.
- **Fondo Interactivo:** Red neuronal dinámica renderizada en Canvas con interacción de mouse.
- **Animaciones de Revelado:** Elementos que aparecen suavemente al hacer scroll.
- **Optimización de Lectura:** Uso de tipografías modernas (Inter y Outfit).
- **Responsive Design:** Totalmente adaptado para móviles, tablets y escritorio.

## Estructura del Proyecto
- `index.html`: Estructura semántica y optimización SEO.
- `style.css`: Estilos visuales refinados y sistema de diseño.
- `main.js`: Lógica del fondo animado, efectos de navegación y animaciones.

## Cómo Visualizar
Simplemente abre el archivo `index.html` en cualquier navegador moderno.
